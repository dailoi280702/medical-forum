## For development:

- Clone this project.
- Run command in project root to install dependencies:

```
# for npm user:
npm i

# for yarn user:
yarn
```

- To run:

```
# for npm user:
npm run dev

# for yarn user:
yarn dev
```

- Install [Eslint](https://eslint.org/) and [Prettier](https://prettier.io/) (_if you use vs code, install extensions instead_)
